# FrontEnd

Aplicación web del sistema comercial multisucursal orientada a la gestión operativa y administrativa de la empresa.

## Tecnologías previstas
- React JS
- JavaScript
- Vite
- React Router
- Axios
- CSS o framework de UI por definir

## Objetivo
Construir una interfaz clara, rápida y mantenible para los usuarios del sistema.

## Módulos previstos
- Inicio de sesión
- Panel principal
- Gestión de productos
- Gestión de sucursales
- Inventario
- Ventas
- Usuarios
- Reportes

## Roles previstos
- Administrador general
- Administrador de sucursal
- Vendedor

## Buenas prácticas a seguir
- Componentes reutilizables
- Separación entre vistas, componentes, servicios y hooks
- Manejo centralizado de peticiones HTTP
- Rutas protegidas por rol
- Validación de formularios
- Nombres consistentes para archivos y componentes
